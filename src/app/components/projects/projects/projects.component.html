Sistema de locação

Descrição: Aplicação web composta por um backend em Java com Spring Boot e um frontend em Angular. O backend é responsável por toda a lógica de negócio, autenticação via JWT, persistência dos dados em MySQL e segurança da aplicação. O frontend oferece uma interface responsiva e intuitiva, facilitando a navegação e a interação do usuário com as funcionalidades da API.


Tecnologias: Java, Spring Boot, Spring Security, Spring Data JPA, JWT, MySQL, Docker, Angular.

Plataforma Cursos

Descrição: Aplicação web desenvolvida utilizando Angular no frontend e um backend baseado em microsserviços com Java e Spring Boot. O sistema permite que usuários realizem cadastro, inscrição em cursos, pagamentos de matrícula e acesso às aulas em vídeo. Conta também com uma área administrativa onde é possível gerenciar cursos, aulas e usuários. O backend é composto por múltiplos microsserviços, utilizando API Gateway, service discovery (Eureka), autenticação via JWT, comunicação entre serviços com OpenFeign e mensageria com RabbitMQ. Integrações externas foram realizadas com o Mercado Pago (Checkout Pro) para processamento de pagamentos e com a API do Gumlet para upload e gerenciamento dos vídeos dos cursos.


Tecnologias: Java, Spring Boot, Spring Security, Spring Data JPA, Spring Cloud Gateway, Spring Cloud Eureka, RabbitMQ, OpenFeign, MySQL, Docker, Mercado Pago SDK, API Gumlet, Angular.
