Plataforma Cursos Front-End

Desenvolvi o front-end de uma plataforma de cursos utilizando Angular. O sistema permite que usuários realizem cadastro na plataforma, inscrição em cursos e acesso às aulas. Possui também uma área administrativa exclusiva, onde administradores podem gerenciar cursos, aulas e usuários.
A aplicação se integra com um backend baseado em microserviços, utilizando autenticação JWT e comunicação via API Gateway.

  • Angular

Plataforma Cursos Microserviços Back-End

Backend desenvolvido (para estudo de microserviços) com Java + Spring Boot utilizando arquitetura de microserviços. O projeto inclui Eureka Server, API Gateway, autenticação via JWT, comunicação entre serviços com OpenFeign e RabbitMQ, persistência com MySQL + JPA, e um serviço de e-mail para recuperação de senha. Também integra com o Mercado Pago Checkout Pro para pagamentos de matrícula e com a API do Gumlet para upload e gerenciamento dos vídeos dos cursos.
  • Java
  • MySQL
  • Spring-Boot
  • Spring Boot Security
  • Spring Cloud Gateway
  • Spring Cloud Eureka
  • RabbitMQ
